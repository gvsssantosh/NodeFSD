What does HTML stand for?

HTML stands for HyperText Markup Language.

What is the basic structure of an HTML document?

The basic structure of an HTML document consists of <html>, <head> and <body> tags.

What is the purpose of the <head> tag in HTML?

The <head> tag contains meta-information about the HTML document, such as its title, links to stylesheets, and scripts.

How do you create a hyperlink in HTML?

You create a hyperlink using the <a> tag, with the href attribute specifying the URL you want to link to.

What is the role of the <div> tag in HTML?

The <div> tag is a container element that is used to group other HTML elements together and apply styling to them.

What is the purpose of the meta tag in HTML?

The meta tag provides meta-information about the HTML document, such as character encoding, keywords, and description for search engines.
